Steps of inserting into a RedBlack Tree

  1. Step 1 - Check whether tree is Empty.
  2. Step 2 - If tree is Empty then insert the newNode as Root node with color Black and exit from the operation.
  3. Step 3 - If tree is not Empty then insert the newNode as leaf node with color Red.
  4. Step 4 - If the parent of newNode is Black then exit from the operation.
  5. Step 5 - If the parent of newNode is Red then check the color of parentnode's sibling of newNode.
  6. Step 6 - If it is colored Black or NULL then make suitable Rotation and Recolor it.
  7. Step 7 - If it is colored Red then perform Recolor. Repeat the same until tree becomes Red Black Tree.
